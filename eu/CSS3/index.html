CSS3

O que é CSS?

Bom, Imagine um jornal, ao abrir o jornal voce verá que existe imagens, textos, e se for online, exitem videos e audios. Essa parte é o HTML.

Entretanto, se existir somente isso em um site, o site fica Confuso, sem estetica e design e logo deixaria o usuario enjoado e faria ele sair do site. Mas pra contornar esse problema foi criado o CSS. O CSS3 é uma linguagem de Estilizaçao e consegue mudar Caracteristicas do site, como tamanho do site, Largura e Altura, mudar as cores do site, e até fazer animaçoes!!

Mas, Como eu posso criar arquivo o CSS

  • Passo n1°: Crie o arquivo "Style.css"
  • Passo n2°: No HTML use abaixo da "title" do html a tag <"link rel="stylesheet" href="Coloque Aqui a pasta do arquivo">

Pronto, vc aprendeu a criar um arquivo CSS.

Como modificar Objetos/Tags?

Para isso, voce ira ter que escolher a tag que vc quer mudar e modificar usando os comandos abaixo:

    Alterar Cores

  • Background-color: Isso ira muadar a cor de fundo de um objeto/tag.
  • color: Isso Ira mudar a cor do objeto/tag

    • Alterar Medidas

  • width:Isso ira mudar a Largura do objeto/tag
  • Height: Isso irá mudar a Altura de um objeto/tag

    • Adicionar Sombras e bordas

  • Border: Adiciona uma Borda a um objeto/tag
  • Border Radius: Configura o tamanho do raio da borda
  • Border color: Muda a cor da borda

Existem Muitas Tags, porem as que eu mais uso Sao as tags Acima.

no codigo ela ficara assim:

Div.exemplo{
Background-color: Black;
color: green;
border-color: white;
box-shadow: 2px 3px 5px var(--cor4);
border-bottom-left-radius: 20px;
border-top-right-radius: 20px;
padding: 15px;
margin-top: 15px;
margin-botton: 15px;
}

Esse é o mesmo codigo que usei pra mudar a cor, borda e raio dessa div acima.
