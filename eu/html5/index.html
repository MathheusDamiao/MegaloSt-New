O que é o HTML?

HTML é uma Linguagem de Marcaçao de Hipertexto, nao e'uma linguagem de progamacao, mas sim uma linguagem de desenvolvimento.tudo o que é Conteudo, como fotos, textos, videos, audios, e até mapas sao feitos pelo html.

Como o html funciona?

O HTML Usa as Tags para a identificaçao de um titulo, paragrafo, tabelas etc...

Essas sao tags que servem para formataçao de texto

p, Que serve para escrever um paragrafo


h1, serve para escrever um titulo


h2, serve para escrever um subtitulo


h3, que é o subtitulo do subtitulo

Code, Faz as letras ficarem assim


q, faz uma palavra ficar entre aspas Assim


Blockquote, serve para Dar espaçamento em uma citaçao


cite, mostra a origem da citaçao a partir do url


abbr, serve para mostrar o significado de uma abreviaçao.tipo isso


bdo, Serve para inverter a direçao das palavras


ol, é o pai da tag li, se chama lista ordenada


li, serve para fazer listas


ul, é a pai das listas, porem se chama listas nao ordenadas


a, serve para criar links


dt, serve para adicionar um termo, e trabalha em conjunto com a tag dd


picture, serve para adicionar uma imagem


audio, serve pra adicionar audio


video, serve pra adicionar uma video


Existem varias outras tags, porem essas sao as mais importantes.
